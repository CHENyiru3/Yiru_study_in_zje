# 贡献指南
同学，你终于也要迈出贡献到开源社区的这伟大一步了！感谢你的支持！在此之前，请先阅读：

!!
1. **假设我的文件格式完全是markdown**：那么网站原生支持显示；请确保一件事情，你的图像不是完全存在本地。我的工作流是图像会自动上传到某一个图像网站并且自动编码成网站链接，所以不影响上传；否则，请看第二条。

2. **假设我的文件格式完全是pdf**：非常好，这个网站也原生支持。Markdown如果图像都在本地，也请先转换成pdf格式，有很多网站可以做这件事，Obsidian也可以。只要是Pdf都可以！网站之后会自动压缩pdf，通过网页直接可以下载zip合集。

3. **假设你的文件格式很奇怪**：暂时没有排查所有格式，但是思维导图，OneNote，等比较软件独有的格式基本不能网页直接查看。此时最好的方法有两个，第一个，是直接上传到你自己的某一个云盘，确保共享链接后，可以写一个简单的概述，然后把概述和链接发送给我。第二个，是和我联系，文件传输给我后，我会帮你上传到我的Google Drive（通常更推荐，因为我会进行统一整理）。

4. **我想贡献课上的作业和期末考试内容，可以吗**：很遗憾，这个其实是一个学校不太允许的事，为了网站持续运行（本人xhs已经被E开盒了），暂时不支持公开。主要欢迎大家分享期末总结的笔记，或者课程中学习总结的笔记；零碎的上课的记录等，也其实不是很推荐，以为信息不集中的话比较难以用于复习。

5. **我很想贡献，但是不知道该怎么做**：没事，欢迎联系我，有任何不明白的地方或者你想分享课程之外的知识笔记！

6. **课程笔记重复了**： 是的，随着贡献者越来越多，这是肯定会发生的。但是我相信没有任何人的笔记是完美的，反过来说，不管谁提供笔记，总能带来新的视角！网站已经在尽力建设分类体系。

!!
